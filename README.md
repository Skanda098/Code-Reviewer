<h1> CODELINGO </h1>

<h3>Requirements : </h3>
Python 3.8+ installed on your system. <br><br>

<h3>Procedure :</h3>
Open your terminal or command prompt and navigate to the project folder. <br> <br>
Then type the following commands <br><br>

``` python -m venv venv ``` <br><br>
``` venv\Scripts\activate ``` <br><br>
``` pip install fastapi uvicorn pydantic deep-translator ```<br><br>
```pip install deep-translator``` <br><br>
```uvicorn main:app --reload```

<h1></h1>
Do not close the old terminal <br>
Open a new terminal or command prompt and navigate to the same project folder of code reviewer using cd command. <br> <br>

```python -m http.server 5500```<br> <br>
Go to your browser open this exact URL : 

```http://localhost:5500```
<br><br>
